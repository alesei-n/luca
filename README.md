What Is Luca?
-------------
Luca is a component / container library for Backbone.js which
encompasses all of the best practices for laying out a large
app with many nested views and containers.

To run the sandbox:

* `bundle install`
* `rackup`

visit http://localhost:9292

Using it with Rails
-------------------
```ruby
# Gemfile
gem 'luca'
```

javascript manifest

```javascript
//= require luca-ui
```

stylesheet manifest

```css
/*
 *= require luca-ui
*/
```
